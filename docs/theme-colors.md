# Nexterm Theme Colors

Nexterm 共有 **23 种主题配色**，分为 12 种暗色主题和 11 种亮色主题。每个主题包含 25 个 UI 颜色属性及 1 个配套 xterm 终端字体颜色方案（ANSI 16 色 + 光标/选区/搜索色）。

---
